SharePoint External User Manager - Preview

Web Part Features

  • ✅ Modern React component with Fluent UI
  • ✅ Responsive DetailsList for external libraries
  • ✅ CommandBar with action buttons (Add, Remove, Manage Users)
  • ✅ Mock data service with 5 sample libraries
  • ✅ Permission-based styling and validation
  • ✅ Modular architecture for backend integration
  • ✅ TypeScript support with proper type definitions
  • ✅ SCSS module styling

Mock Data Preview

The component displays 5 sample external libraries:

  • Marketing Documents - 5 external users, Contribute permissions
  • Project Alpha Resources - 3 external users, Read permissions
  • Vendor Collaboration Hub - 8 external users, Full Control permissions
  • Customer Support Files - 2 external users, Read permissions
  • Training Materials - 4 external users, Contribute permissions

Action Placeholders

The following actions are implemented as placeholders that show alerts:

  • Add Library - Placeholder for adding new external libraries
  • Remove - Enabled when libraries are selected
  • Manage Users - Enabled when exactly one library is selected
  • Refresh - Reloads the mock data with loading animation
